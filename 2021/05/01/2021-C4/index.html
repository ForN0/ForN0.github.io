Part01: 挑战书

尊敬的赛组老师您好：

        信息网络时代已然到来，云+安全成为社会各行各业的热题。没有网络安全就没有国家安全，作为计算机系的青年一代人，本着将理论知识升华为项目实战的初心：

        谨以此向2021年中国高校计算机大赛——网络技术挑战赛-A系列发起挑战！



Part02: 设计背景

        从2010年，在信息化“十二五”规划中提出的“智慧校园”。到2018年，国家标准化《智慧校园总体框架》的发布。据网络问卷调研不完全数据显示，时至今日，智慧校园这个以云和IoT技术依托将教学、科研、管理和校园生活一体化的已经融入在近90%的高等院校中。



Part03: 项目蓝图

前言说明：
        本项目是一套模拟高等院校网络系统环境进行的综合实战项目挑战方案，旨在将网络设计规划、系统设计规划、校园云设计规划从3个方向同时展开并进行综合实战，且将整体安全系统意识贯穿在整个 C4 网络综合项目实践挑战之旅中。





Part04: 规划要点

网络设计规划

· 区域概述

Untrust(非信任区)：在校园出入口放置双活防火墙NGFW-USG6000系列，阻隔恶意流量。

Trust(信任区)：在不同子区域设置虚拟网络隔离，放置多点AP无线设备提供WiFi信号通过AC管理，同时实现校园无线流量身份鉴权识别，流量监控管理以便将非法流量下线处置，保证无线网上冲浪的安全性。

DMZ(军事区)：放置防火墙等安全设备，进行VPN配置，方便远程与异地办公。

· 无线校园网

        随着现代通信技术发展，无线网络越来越广泛地应用，基于计算机局域网技术的无线校园网得到推广。无线校园网以其具备的便携灵活性满足了师生们在随时随地移动办公学习的迫切需求，弥补了传统有线网络布线复杂、高成本、扩展性弱的网络盲点。

        技术点关键点：WVLAN、AP、AC

· eSight系统

        部署eSight作为统一运维管理结点，降低运维成本，提升网络运维效率。

系统设计规划

· Linux简介

        Linux 是一个自由、免费、开源的操作系统，是不受任何商品化软件版权制约的、全世界都能使用的类Unix兼容产品。因安全性稳定性极高，本项目系统侧主要采用Linux作为操作系统，实现整个项目服务上的架构。

· 系统服务设计

校园官网-WEB：采用LNMP架构(Linux+Nginx+MySQL+PHP)。

校园网盘-NextCloud：一款开源的优秀网盘文件管理系统，可以为全校同学提供定量的存储资源空间。

校园码云-Gitee：一款开源的优秀代码托管系统，可以为提供极佳的校园代码交友体验。

校园天眼-Prometheus+Granafa：一款开源可自定制二次开发的开源监控系统，能用于统一的设备监管审计运维。

镜像源站：收集更新常用镜像及安装包，提供统一的快速软件部署服务源，高速便捷内网且方便外网展示开源精神。

· 虚拟化与集群技术

        虚拟化通常指通过软件模拟的具有完整硬件系统功能的、运行在一个完全隔离环境中的完整计算机系统。系统设计将与云设计相结合，选用OpenStack技术对实有的硬件物理服务器资源进行虚拟化，在云上创建不同网段对应不同片区的资源，实现对全局资源的统一管理和监控。更进一步的，可以在云上建Docker，打镜像，实现资源的灵活管理，提高可用性。且考虑到稳定性和实用性，后期会考虑采用分布式集群方式部署相关的系统业务。

校园云设计规划

· 设计概要

        主要基于华为云产品的选用。

        校园云设计规划由桌面管理软件FusionAccess、云操作系统软件FusionCompute和FusionManager（选）云监控三部分组成，并结合主流OpenStack技术将底层环境虚拟化后将系统规划设计部分进行上云。

        方案计划，首先在CNA（计算节点代理）的基础上部署VRM（虚拟资源管理）建立FusionCompute平台；其次，在已有FusionCompute的环境基础之上，搭建FusionAccess平台，通过该平台制作模板机为教学环境分发相应云桌面；再次FusionManager（选）同样在FusionCompute的基础上搭建，通过统一的接口对整个云平台进行维护、监控。

· 方案价值

数据上移，信息安全。桌面云环境可打破传统桌面环境的物理限制，做到终端和信息的分离，使得桌面和数据在后端集中存储、处理。

高效维护，自动管控。拥有一键式维护工具，大大降低维护成本，维护效率提高至四倍以上。

应用上移，业务可靠。业务和应用都在数据中心处理，机房保障系统可以包保障99.9%的业务员稳定性，保持业务的连续性。

无缝切换，移动办公。数据和桌面都集中保存、运行在数据中心，因此可以不中断应用的运行，无缝切换办公地点，做到真正的移动办公。

资源弹性，复用共享。所有资源都在数据中心，可以时限资源的集中管控，弹性调度。

· Fusion系列套件说明

-FusionCompute

虚拟化基础引擎，它采用了虚拟化技术对硬件进行虚拟化处理，主要包含计算资源、存储资源、网络资源虚拟化。通过提供的图形界面可以直观了解资源的使用情况并且对这些虚拟资源进行统一管理、监控。

-FusionAccess

基于云平台的一种虚拟桌面应用，可以为提供学生分发、回收云桌面等功能。学生体验到的虚拟机与普通电脑无异，其使用的物理设备为瘦终端，在此可以方面减少购买物理设备的开销。

-FusionManager

云管理平台，其核心为云服务自动化和资源智能运维，为提供云数据中心的管理。FusionManager可以接入和管理物理资源、虚拟化资源、桌面云以及云存储资源。其中基于角色访问控制的策略、为资源提供虚拟化、虚拟化资源的管理监控、容灾备份等功能。

[附] FAQ TOP8·说明资料

1.关于WVLAN说明

无线虚拟局域网是指在无线局域网的基础上，采用网络管理软件构建的可跨越不同网段、不同网络的端到端的逻辑网络。它是以无线方式构建的局域网WLAN，或者说，不用线缆将设备相连的局域网络。无线虚拟局域网，通过网络管理软件，在无线局域网的基础上建立一个逻辑网络，这个逻辑网络中的工作站和服务器可以不受地理位置的限制，但同样具有物理局域网的功能和特点。

2.关于无线AP和AC控制器说明

无线AP（Access Point）

即无线接入点，它用于无线网络的无线交换机，也是无线网络的核心。无线AP是移动计算机用户进入有线网络的接入点，主要用于宽带家庭、大楼内部以及园区内部，可以覆盖几十米至上百米。无线AP（又称会话点或存取桥接器）是一个包含很广的名称，它不仅包含单纯性无线接入点（无线AP），同样也是无线路由器（含无线网关、无线网桥）等类设备的统称。

AC控制器（Access Control）

它是指无线接入控制服务器(AC)， 接入控制器(AC) 无线局域网接入控制设备，负责把来自不同AP的数据进行汇聚并接入Internet，同时完成AP设备的配置管理、无线用户的认证、管理及宽带访问、安全等控制功能。

3.关于eight平台说明

esight系统提供存储、服务器、应用、交换机、路由器、防火墙、WLAN、PON网络、无线宽带集群设备、视频监控、IP话机、视讯设备等多种设备的统一管理，支持多厂商设备统一视图、资源，拓扑、故障、性能以及智能配置功能，同时为客户提供第三方设备的定制能力与告警北向接口，帮助客户打造专属的统一管理系统，降低运维成本，提升运维效率。

4.为何选用Linux系统

· Linux系统可用于开源用途，通过开源，使用者不仅可以查看Linux内核的代码，还可以对代码进行修改和搭建。

· 使用Linux系统的用户很少会遇到系统崩溃的情况，甚至在运行多年也不会出现重大事故，其稳定性可想而知。

· 用户可以对Linux系统进行自定义，通过编程接口，将自己开发的工具和程序添加到该系统中，以追求更符合自己操作要求的界面。

· Shell作为Linux系统中最强大的组件，完全可以让运行的程序与内核进行交互。

· 相对Windows来说，Linux对硬件的需求比较低，也不需要频繁地对硬件进行升级更新。

· 一般情况，Linux Server只有管理员或特定用户才有访问内核的权限，所以在安全方面来说，受到攻击的次数会相对较低。

· Linux在使用时是免费的，所以成本会下降很多，即使是购买特定的Linux花费的价格也比其他系统和软件要低。

· 与Windows Servers不同，Linux Servers可以在不重启服务器的情况下，自由地对系统进行变更，无需购买其他版本才能使用某些特定的功能。

5.关于LNMP的说明

· LNMP代表的就是：Linux系统下Nginx+MySQL+PHP这种网站服务器架构。

· Linux是一类Unix计算机操作系统的统称，是目前最流行的免费操作系统。代表版本有：debian、centos、ubuntu、fedora、gentoo等。

· Nginx是一个高性能的HTTP和反向代理服务器，也是一个IMAP/POP3/SMTP代理服务器。

· Mysql是一个小型关系型数据库管理系统。

· PHP是一种在服务器端执行的嵌入HTML文档的脚本语言。

这四种软件均为免费开源软件，组合到一起，成为一个免费、高效、扩展性强的网站服务系统。

6.关于NEXTCLOUD的说明

Nextcloud 是一套用于创建网络硬盘的客户端－服务器软件。其功能与 Dropbox 相近，但 Nextcloud 是自由及开放源代码软件，每个人都可以在私人服务器上安装并运行它。与 Dropbox 等专有服务相比，Nextcloud 的开放架构让用户可以利用应用程序的方式在服务器上新增额外的功能，并让用户可以完全掌控自己的数据。

7.关于Gitee的说明

Gitee 是开源中国推出的基于 Git 的代码托管服务。目前已经有注册开发者 600 万，托管超过 1500 万代码仓库。Gitee 包括三个版本，分别是：社区版、企业版和高校版。企业版已为超过 180000 家企业进行服务，高校版已为超过 2000 所高校提供服务。

8.关于Prometheus、Grafana说明

Prometheus 是一套开源的系统监控、报警、时间序列数据库的组合，最初有 SoundCloud 开发的，后来随着越来越多公司使用，于是便独立成开源项目。我们常用的 Kubernetes 容器集群管理中，通常会搭配 Prometheus 一起来进行监控。Prometheus 基本原理是通过 Http 协议周期性抓取被监控组件的状态，而输出这些被监控的组件的 Http 接口为 Exporter，现在各个公司常用的 Exporter 都已经提供了，可以直接安装使用，如 haproxy_exporter、blockbox_exporter、mysqld_exporter、node_exporter 。

Grafana 是一个可视化仪表盘，它拥有美观的图标和布局展示，功能齐全的仪表盘和图形编辑器，默认支持 CloudWatch、Graphite、Elasticsearch、InfluxDB、Mysql、PostgreSQL、Prometheus、OpenTSDB 等作为数据源。我们可以将 Prometheus 抓取的数据，通过 Grafana 优美的展示出来，非常直观。
